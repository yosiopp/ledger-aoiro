# 勘定科目一覧

## 勘定科目の大分類

このプロジェクトでは、勘定科目の大分類に **hledger のタイプコードと一致する大文字1文字**（A, L, E, R, X）を使用しています：

| 勘定科目タイプ | コード    | 日本語 | 借方 | 貸方 |
| -------------- | --------- | ------ | ---- | ---- |
| **A**          | Asset     | 資産   | `+`  | `-`  |
| **L**          | Liability | 負債   | `-`  | `+`  |
| **E**          | Equity    | 純資産 | `-`  | `+`  |
| **R**          | Revenue   | 収益   | `-`  | `+`  |
| **X**          | Expense   | 費用   | `+`  | `-`  |

**例：** `A:銀行:事業用`、`X:消耗品費`、`R:売上`

借方・貸方の詳しい説明は [複式簿記の基礎ガイド](guide.md) を参照してください。
