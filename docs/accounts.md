# 勘定科目一覧

## 勘定科目の命名規則

このプロジェクトでは、勘定科目の大分類に **hledger のタイプコードと一致する大文字1文字**（A, L, E, R, X）を使用しています：

| 勘定科目タイプ | コード    | 日本語 |
| -------------- | --------- | ------ |
| **A**          | Asset     | 資産   |
| **L**          | Liability | 負債   |
| **E**          | Equity    | 純資産 |
| **R**          | Revenue   | 収益   |
| **X**          | Expense   | 費用   |

**例：** `A:銀行:事業用`、`X:消耗品費`、`R:売上`

この命名規則により：

- 入力が簡潔（1文字のみ）
- hledger の `type:` ディレクティブと完全一致
- `balancesheet`、`incomestatement` などの便利機能が正しく動作
