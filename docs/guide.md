# 複式簿記の基礎ガイド

このガイドでは、青色申告に必要な複式簿記の基本概念を説明します。

## 複式簿記とは

**複式簿記**（double-entry bookkeeping）は、すべての取引を「借方」と「貸方」の2つの側面から記録する記帳方法です。

### 単式簿記との違い

- **単式簿記**：お小遣い帳や家計簿のように、入金・出金だけを記録

  ```
  1/10  給料     +300,000円
  1/15  家賃     -100,000円
  1/20  食費      -30,000円
  ```

- **複式簿記**：お金の動きと同時に、その原因も記録
  ```
  1/10  給料が入った → 銀行預金が増えた（資産↑）、売上が発生した（収益↑）
  1/15  家賃を払った → 銀行預金が減った（資産↓）、地代家賃が発生した（費用↑）
  ```

複式簿記を使うことで、**財産の状態（貸借対照表）** と **儲けの状況（損益計算書）** の両方を正確に把握できます。

## 借方と貸方の基本

すべての取引は **借方（かりかた / Debit）** と **貸方（かしかた / Credit）** の2つの側面から記録します。

### 基本ルール

```
借方（左側）                    貸方（右側）
─────────────────────────────────────────
         金額の合計が必ず一致
```

**重要：** どんな取引でも、借方の合計金額と貸方の合計金額は必ず一致します。これを **貸借平均の原則** と呼びます。

### 5つの勘定科目タイプ

会計では、すべての勘定科目を5つのタイプに分類します：

| タイプ               | コード | 説明                       | 借方で増加 | 貸方で増加 |
| -------------------- | :----: | -------------------------- | ---------- | ---------- |
| **資産** (Assets)    |  `A`   | 現金、預金、売掛金など     | ✓          |            |
| **負債** (Liability) |  `L`   | 借入金、買掛金、未払金など |            | ✓          |
| **純資産** (Equity)  |  `E`   | 元入金、繰越利益など       |            | ✓          |
| **収益** (Revenue)   |  `R`   | 売上、雑収入など           |            | ✓          |
| **費用** (Expense)   |  `X`   | 経費全般                   | ✓          |            |

### 仕訳のルール

勘定科目のタイプによって、増加・減少の記録位置が決まっています：

#### 資産（A）・費用（X）

| 状態           | 借方（左側） | 貸方（右側） |
| -------------- | ------------ | ------------ |
| **増えたとき** | 資産 / 費用  |              |
| **減ったとき** |              | 資産 / 費用  |

**例：**

- 現金（資産）が増えた → 借方に記入
- 通信費（費用）が発生した → 借方に記入
- 銀行預金（資産）が減った → 貸方に記入

#### 負債（L）・純資産（E）・収益（R）

| 状態           | 借方（左側）         | 貸方（右側）         |
| -------------- | -------------------- | -------------------- |
| **増えたとき** |                      | 負債 / 純資産 / 収益 |
| **減ったとき** | 負債 / 純資産 / 収益 |                      |

**例：**

- 売上（収益）が発生した → 貸方に記入
- クレジットカード（負債）が増えた → 貸方に記入
- 借入金（負債）を返済した → 借方に記入

## 実践的な仕訳例

### 例1：現金売上（収入があった）

**取引：** クライアントから現金で50,000円の報酬を受け取った

**何が起きた？**

- 現金という資産が増えた（資産↑）
- 売上という収益が発生した（収益↑）

**仕訳：**

| 借方          | 貸方          |
| ------------- | ------------- |
| 現金 50,000円 | 売上 50,000円 |

**hledger記法：**

```ledger
2026/01/15 * クライアントA 案件報酬
    A:現金                    50000 JPY
    R:売上
```

> [!NOTE]
> hledgerでは2行目の金額を省略すると自動計算されます

### 例2：経費の支払い（費用が発生）

**取引：** 事務所の通信費5,000円を銀行口座から支払った

**何が起きた？**

- 通信費という費用が発生した（費用↑）
- 銀行預金という資産が減った（資産↓）

**仕訳：**

| 借方           | 貸方                |
| -------------- | ------------------- |
| 通信費 5,000円 | 銀行:事業用 5,000円 |

**hledger記法：**

```ledger
2026/01/20 * インターネット料金
    X:通信費                  5000 JPY
    A:銀行:事業用
```

### 例3：クレジットカードで経費購入

**取引：** 業務用プリンターを85,000円でクレジットカード購入

**何が起きた？**

- 消耗品費という費用が発生した（費用↑）
- クレジットカードという負債が増えた（負債↑）

**仕訳：**

| 借方              | 貸方                      |
| ----------------- | ------------------------- |
| 消耗品費 85,000円 | クレジットカード 85,000円 |

**hledger記法：**

```ledger
2026/02/05 * 業務用プリンター購入
    X:消耗品費              85000 JPY
    L:クレジットカード
```

### 例4：クレジットカード代金の引き落とし

**取引：** 先月のクレジットカード利用分85,000円が銀行口座から引き落とされた

**何が起きた？**

- クレジットカードという負債が減った（負債↓）
- 銀行預金という資産が減った（資産↓）

**仕訳：**

| 借方                      | 貸方                 |
| ------------------------- | -------------------- |
| クレジットカード 85,000円 | 銀行:事業用 85,000円 |

**hledger記法：**

```ledger
2026/03/10 * クレジットカード引き落とし
    L:クレジットカード      85000 JPY
    A:銀行:事業用
```

### 例5：生活費の引き出し（事業主貸）

**取引：** 事業用口座から生活費として100,000円を引き出した

**何が起きた？**

- 事業主貸という純資産が増えた（事業のお金を個人に貸した）
- 銀行預金という資産が減った（資産↓）

**仕訳：**

| 借方               | 貸方                  |
| ------------------ | --------------------- |
| 事業主貸 100,000円 | 銀行:事業用 100,000円 |

**hledger記法：**

```ledger
2026/01/25 * 生活費引き出し
    E:事業主貸              100000 JPY
    A:銀行:事業用
```

> [!NOTE]
> 個人事業主では、事業用のお金を私的に使う場合は「事業主貸」で記録します

### 例6：売掛金の計上と回収

#### 取引1：請求書を発行（売上発生）

**何が起きた？**

- 売掛金という資産が増えた（資産↑）
- 売上という収益が発生した（収益↑）

**仕訳：**

| 借方             | 貸方           |
| ---------------- | -------------- |
| 売掛金 150,000円 | 売上 150,000円 |

**hledger記法：**

```ledger
2026/01/31 * クライアントB 1月分請求
    A:売掛金                150000 JPY
    R:売上
```

#### 取引2：翌月入金された（売掛金回収）

**何が起きた？**

- 銀行預金という資産が増えた（資産↑）
- 売掛金という資産が減った（資産↓）

**仕訳：**

| 借方                  | 貸方             |
| --------------------- | ---------------- |
| 銀行:事業用 150,000円 | 売掛金 150,000円 |

**hledger記法：**

```ledger
2026/02/28 * クライアントB 入金
    A:銀行:事業用           150000 JPY
    A:売掛金
```

## 借方・貸方の覚え方

### 簡単な覚え方：「資産と費用は左、それ以外は右」

| 借方（左側）     | 貸方（右側）     |
| ---------------- | ---------------- |
| 資産の増加       | 資産の減少       |
| 費用の発生       | 負債の増加       |
| 負債の減少       | 純資産の増加     |
| 純資産の減少     | 収益の発生       |
| 収益の減少（稀） | 費用の減少（稀） |

### 実務での考え方

実際に仕訳をするときは、次のステップで考えると簡単です：

1. **何が起きたか？** を2つの側面で考える
   - 例：「パソコンを買った」→「消耗品費が発生した」＋「現金が減った」

2. **それぞれが増えた？減った？** を判断
   - 消耗品費（費用）が発生 → 費用が増加
   - 現金（資産）が減った → 資産が減少

3. **表のルールに従って配置**
   - 費用の増加 → 借方（左側）
   - 資産の減少 → 貸方（右側）

```ledger
2026/01/10 * パソコン購入
    X:消耗品費              50000 JPY  ; 借方（費用の増加）
    A:現金                             ; 貸方（資産の減少）
```

## 複式簿記のメリット

1. **自己検証機能**：借方と貸方が常に一致するため、記帳ミスを発見しやすい
2. **財務状況の把握**：資産・負債の状態と、利益の状況を同時に管理できる
3. **青色申告特別控除**：65万円の控除を受けるには複式簿記が必須
4. **経営判断の精度向上**：正確な財務データに基づいた意思決定が可能

## よくある質問

### Q1: なぜ「借方」「貸方」という名前なのか？

これは歴史的な用語です。元々は商人が「貸し借り」を記録していたことに由来します。現代では単に「左側」「右側」という位置を示す用語と考えて問題ありません。

### Q2: 借方と貸方、どちらに書くか毎回迷います

最初は誰でも迷います。繰り返し練習することで、自然と身につきます。このプロジェクトでは `./lgr check` コマンドで貸借が一致しているか確認できるので、間違いがあればすぐに気づけます。

### Q3: 「事業主貸」「事業主借」とは？

個人事業主特有の勘定科目です：

- **事業主貸（E:事業主貸）**：事業のお金を個人的に使ったとき（生活費など）
- **事業主借（E:事業主借）**：個人のお金を事業に入れたとき

法人の「資本金」に相当する概念ですが、個人事業主では事業と個人の境界が曖昧なため、このような勘定科目が必要になります。
