# 確定申告ガイド

このドキュメントでは、ledger-aoiro を使った帳簿から青色申告決算書を作成し、確定申告を行う流れを説明します。

## 確定申告のスケジュール

| 時期             | 作業内容                       |
| ---------------- | ------------------------------ |
| 12月末           | 決算整理仕訳の作成             |
| 1月              | 年間集計の確認、決算書の下書き |
| 2月中旬〜3月15日 | 確定申告書の提出（e-Tax推奨）  |

**提出期限：** 翌年2月16日〜3月15日（土日の場合は翌平日）

## 決算処理の流れ

### 1. 年末決算整理（12月31日）

年度末（通常は12月31日）に以下の決算整理仕訳を行います。

#### 減価償却費の計上

```ledger
; ledger/2026/closing.ledger

2026/12/31 * 減価償却費（ノートPC）
    X:減価償却費              37500 JPY  ; 150000円÷4年
    A:備品
```

#### 前払費用の整理

年をまたぐ前払費用がある場合は調整します。

```ledger
2026/12/31 * 翌年分の前払費用
    A:前払費用                30000 JPY  ; 2027年分
    X:通信費  ; 当年の経費から控除
```

#### 売掛金・買掛金の確認

年末時点で未回収の売上、未払の経費を確認します。

### 2. 損益の集計

年間の収益と費用を集計して、事業所得を計算します。

```bash
# 年次集計を実行
./lgr yearly
```

このコマンドで以下が表示されます：

- 収益合計（売上、雑収入）
- 費用合計（各経費科目の内訳）
- 事業所得（収益 - 費用）

### 3. 貸借対照表の確認

年末時点の資産・負債・純資産の残高を確認します。

```bash
./lgr exec -f ledger/accounts.ledger balancesheet
```

## 青色申告決算書の作成

### 必要な書類

1. **青色申告決算書（一般用）** - 4ページ
   - 損益計算書
   - 月別売上・仕入
   - 減価償却費の計算
   - 貸借対照表

2. **確定申告書 第一表・第二表**

### ledger データから決算書への転記

#### 損益計算書（1ページ目）

年次集計の出力を使って各科目を転記します。

```bash
./lgr yearly
```

**収入金額：**

- 売上（収入）金額 → `R:売上` の合計

**経費：**

- 租税公課 → `X:租税公課`
- 水道光熱費 → `X:水道光熱費`
- 旅費交通費 → `X:旅費交通費`
- 通信費 → `X:通信費`
- 消耗品費 → `X:消耗品費`
- 広告宣伝費 → `X:広告宣伝費`
- 地代家賃 → `X:地代家賃`
- 減価償却費 → `X:減価償却費`
- 雑費 → `X:雑費`

**青色申告特別控除：** 65万円（電子帳簿保存またはe-Tax）

**事業所得：** 収入 - 経費 - 青色申告特別控除

#### 貸借対照表（4ページ目）

**資産の部：**

```bash
./lgr exec balance A:
```

- 現金 → `A:現金`
- 預金 → `A:銀行:事業用`
- 売掛金 → `A:売掛金`
- 備品 → `A:備品`

**負債・資本の部：**

```bash
./lgr exec balance L: E:
```

- 買掛金 → `L:買掛金`
- 未払金 → `L:未払金`
- 事業主借 → `E:事業主借`
- 事業主貸 → `E:事業主貸`
- 元入金 → 期首の純資産
- 青色申告特別控除前の所得金額

### CSV エクスポート

表計算ソフトで集計したい場合は CSV でエクスポートできます。

```bash
./lgr export
```

出力ファイル：`output/transactions.csv`

## 確定申告書の提出

### 提出方法

**推奨：e-Tax（電子申告）**

メリット：

- 青色申告特別控除が65万円（書面提出は55万円）
- 郵送・持参が不要
- 控除証明書の添付を省略できる

必要なもの：

- マイナンバーカード
- ICカードリーダー（またはマイナンバーカード読み取り対応スマホ）
- e-Tax ソフトまたは確定申告書等作成コーナー

**その他の方法：**

- 税務署に持参
- 税務署に郵送（期限日の消印有効）

### 提出書類チェックリスト

- [ ] 確定申告書 第一表・第二表
- [ ] 青色申告決算書（4ページ）
- [ ] 各種控除証明書（医療費、保険料、寄付金など）
- [ ] マイナンバー確認書類（書面提出の場合）

## 注意事項

### 家事按分の記録

自宅兼事務所の場合、按分根拠を明確に記録しておきます。

- 地代家賃：床面積比
- 水道光熱費：作業時間比、床面積比
- 通信費：業務使用割合

按分の根拠は [memo/](../memo/) ディレクトリに記録します。

### 領収書の保管

**保存期間：** 7年間（青色申告の場合）

- 電子帳簿保存法に対応した方法で保存
- スキャンデータでも可（要件あり）
- クラウドストレージでバックアップ推奨

### 青色申告の要件

65万円控除を受けるための要件：

1. **複式簿記による記帳** ✅ ledger-aoiro で対応
2. **貸借対照表・損益計算書の作成** ✅ スクリプトで対応
3. **期限内申告**（3月15日まで）
4. **電子帳簿保存またはe-Tax**

## トラブルシューティング

### 数字が合わない

```bash
# 貸借チェック
./lgr check

# 勘定科目の検証
./lgr validate
```

### 経費の分類がわからない

国税庁の「確定申告書等作成コーナー」のヘルプや、タックスアンサーを参照してください。

- [国税庁タックスアンサー](https://www.nta.go.jp/taxes/shiraberu/taxanswer/index2.htm)
- 不明な場合は税務署に電話で相談（確定申告期間中は混雑）

### 初めての確定申告

税理士に相談するか、税務署の無料相談を利用することをお勧めします。
